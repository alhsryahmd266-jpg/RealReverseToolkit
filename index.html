فك التشويف ← تعديل الـ Smali ← إعادة البناء ← التوقيع ← التثبيت

الخطوة 1 — فك التشويف
apktool d target.apk -o output/ --no-res
jadx -d jadx_output/ target.apk
الخطوة 2 — تعديل Smali
.method public isLicensed()Z .registers 2 ; PHANTOM RE - تم الحقن هنا const/4 v0, 0x1 ; return true return v0 .end method
الخطوة 3 — إعادة البناء
apktool b output/ -o modified.apk
الخطوة 4 — إنشاء Keystore
keytool -genkey -v -keystore phantom.jks -alias phantom -keyalg RSA -keysize 4096 -validity 36500 -storepass phantom2024 -keypass phantom2024 -dname "CN=PHANTOM,O=RE,C=EG"
الخطوة 5 — Zipalign
zipalign -v 4 modified.apk aligned.apk
الخطوة 6 — توقيع رقمي
apksigner sign --ks phantom.jks --ks-key-alias phantom --ks-pass pass:phantom2024 --key-pass pass:phantom2024 --out signed.apk aligned.apk
الخطوة 7 — تحقق من التوقيع
apksigner verify --verbose signed.apk
الخطوة 8 — تثبيت
adb install -r signed.apk

💉 Frida Gadget — بدون Root

حقن Frida في أي تطبيق بدون صلاحيات Root بإعادة تحزيمه

1. تحميل Frida Gadget
wget https://github.com/frida/frida/releases/latest/download/frida-gadget-16.0.0-android-arm64.so.xz
xz -d frida-gadget-*.xz && mv frida-gadget-*.so libfrida-gadget.so
2. فك APK
apktool d target.apk -o target_out
3. حقن المكتبة
mkdir -p target_out/lib/arm64-v8a && cp libfrida-gadget.so target_out/lib/arm64-v8a/
4. إضافة System.loadLibrary في smali
const-string v0, "frida-gadget" invoke-static {v0}, Ljava/lang/System;->loadLibrary(Ljava/lang/String;)V
5. إعادة البناء والتوقيع
apktool b target_out -o target_frida.apk && zipalign -v 4 target_frida.apk target_aligned.apk && apksigner sign --ks phantom.jks --ks-pass pass:phantom2024 --key-pass pass:phantom2024 --out target_signed.apk target_aligned.apk
